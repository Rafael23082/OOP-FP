<h1>Car Rental Management System</h1>
This is a Car Rental Mnagement System which can be used by both Employees and Customers.
Employees are able to:
<ul>
  <li>Add Cars</li>
  <li>Delete Cars</li>
  <li>Edit Cars</li>
</ul>

Customers are able to:
<ul>
  <li>Show Cars</li>
  <li>Search Cars</li>
  <li>Compare Cars</li>
  <li>Rent Cars</li>
</ul>
